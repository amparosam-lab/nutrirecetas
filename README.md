# NutriRecetas

Recetas personalizadas con menú semanal y lista de compra compartida.

## Instalación

```bash
npm install
```

## Desarrollo

```bash
npm start
```

Abre [http://localhost:3000](http://localhost:3000) en tu navegador.

## Variables de entorno

Copia `.env.example` a `.env.local` y completa con tus credenciales de Firebase.

## Despliegue

```bash
npm run deploy
```
