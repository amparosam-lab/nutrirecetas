🚀 FLUJO DE TRABAJO:

1. LOCAL (Tu computadora)
   → npm install
   → npm start
   → Desarrollo en http://localhost:3000

2. GITHUB (Repositorio)
   → git push origin main
   → Código en la nube

3. GITHUB PAGES (Hosting)
   → npm run deploy
   → App en vivo en https://tu-usuario-github.github.io/nutrirecetas

4. FIREBASE (Base de datos)
   → React se conecta automáticamente
   → Datos guardados en tiempo real
